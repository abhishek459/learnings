- Floats:
  1. Float: Used to float elements to the left or right.
  2. Commonly used to wrap text around floated elements.
  3. Spacing: Use `margin-right` for `float: left` and `margin-left` for `float: right` to create proper spacing between the floated element and text.
  4. Containerizing floated elements: Use `display: flow-root;` to properly contain floated elements and the wrapped text.
