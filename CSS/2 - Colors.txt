- Colors:
  1. Color Names: `blue`, `black`, `pink`, etc.
  2. RGB: `rgb(255, 0, 0)` (Red, Green, Blue; max 255).
  3. RGBA: `rgba(255, 0, 0, 0.5)` (adds transparency using alpha).
  4. Hexadecimal: `#000000` (black) or shorthand `#000`.
  5. HSL: `hsl(224, 73%, 49%)` (Hue, Saturation, Lightness).
  6. Contrast: Use a contrast checker for accessible background and foreground colors.
